# Lecture-04-responsive

## Features implemented
- Theme toggle: click **Switch to dark** to toggle dark/light themes (also press `t` as a shortcut).
- Click counter: click **Click me** to increment the counter (value shown next to button).

## How to test
1. Open `index.html` in a browser.
2. Click the **Switch to dark** button to toggle theme; check browser console for theme logs.
3. Click the **Click me** button to increment the counter; watch the number and console logs.
